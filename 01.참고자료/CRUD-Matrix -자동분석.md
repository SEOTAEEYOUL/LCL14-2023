# [전사 TCL] Python 을 활용한 CRUD Matrix 자동분석 연구

## [상세 주제]
- Python 으로 MES 소스를 분석
  - UI 와 SQL 을 매핑하고 DB 의 Object 와 SQL 를 매핑

- SQL Parser 를 활용
  - Query 및 SP 등을 parsing 하여 CRUD 를 분석

- openpyxl 패키지 활요
  - 최종 결과를 엑셀문서 출력(UI 및 DB Object & Table CRUD Matrix)
